## Transposição e Redimensionamento de Arrays: Manipulação da Forma dos Dados

Em computação científica, é muito comum precisarmos de alterar a "forma" (shape) ou a orientação dos nossos arrays. O NumPy oferece ferramentas poderosas para **transposição** (trocar linhas por colunas) e **redimensionamento** (mudar o número de linhas/colunas/dimensões).
"""

# Célula de Código
import numpy as np

# Célula de Markdown
"""
### 1. Transposição de Arrays (`.T` ou `np.transpose()`)

A transposição de uma matriz é o processo de trocar as suas linhas pelas suas colunas. Se uma matriz tem a forma `(m, n)`, a sua transposta terá a forma `(n, m)`.

Podemos usar o atributo `.T` ou a função `np.transpose()`.

**Exemplo:**
"""

# Célula de Código
matriz_original = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
print(f"Matriz Original (forma {matriz_original.shape}):\n{matriz_original}")

matriz_transposta = matriz_original.T
print(f"\nMatriz Transposta (forma {matriz_transposta.shape}):\n{matriz_transposta}")

# Também funciona com np.transpose()
matriz_transposta_func = np.transpose(matriz_original)
print(f"\nMatriz Transposta (com np.transpose()):\n{matriz_transposta_func}")

# Célula de Markdown
"""
**Experimentem:**

1.  Criem uma matriz 3x2.
2.  Transponham-na e imprimam a matriz original e a transposta, incluindo as suas formas.
"""

# Célula de Código (para o exercício)
minha_matriz_3x2 = np.array([
    [10, 20],
    [30, 40],
    [50, 60]
])
print(f"Minha Matriz 3x2 (forma {minha_matriz_3x2.shape}):\n{minha_matriz_3x2}")

minha_matriz_3x2_transposta = minha_matriz_3x2.T
print(f"\nMinha Matriz 3x2 Transposta (forma {minha_matriz_3x2_transposta.shape}):\n{minha_matriz_3x2_transposta}")


# Célula de Markdown
"""
---

### 2. Redimensionamento de Arrays (`.reshape()`)

O método `.reshape()` permite alterar a forma de um array sem alterar os seus dados. O número total de elementos no array deve permanecer o mesmo.

**Sintaxe:** `array.reshape(nova_forma)`

* `nova_forma` é um tuplo que especifica as novas dimensões.
* Podemos usar `-1` como uma das dimensões, e o NumPy irá calcular automaticamente essa dimensão para nós.

**Exemplo:** Converter um vetor em uma matriz.
"""

# Célula de Código
vetor_1d = np.arange(1, 13) # Array de 1 a 12
print(f"Vetor 1D (forma {vetor_1d.shape}):\n{vetor_1d}")

# Converter para uma matriz 3x4
matriz_3x4 = vetor_1d.reshape((3, 4))
print(f"\nMatriz 3x4 (forma {matriz_3x4.shape}):\n{matriz_3x4}")

# Converter para uma matriz 4x3 (NumPy calcula a segunda dimensão)
matriz_4x3 = vetor_1d.reshape((4, -1))
print(f"\nMatriz 4x3 (forma {matriz_4x3.shape}):\n{matriz_4x3}")

# Célula de Markdown
"""
**Experimentem:**

1.  Criem um vetor com 15 elementos (ex: `np.arange(15)`).
2.  Redimensionem este vetor para uma matriz 3x5.
3.  Redimensionem a mesma matriz 3x5 para uma matriz 5x3.
"""

# Célula de Código (para o exercício)
meu_vetor_15 = np.arange(15)
print(f"Meu vetor 15 elementos (forma {meu_vetor_15.shape}):\n{meu_vetor_15}")

matriz_3x5 = meu_vetor_15.reshape((3, 5))
print(f"\nMatriz 3x5 (forma {matriz_3x5.shape}):\n{matriz_3x5}")

matriz_5x3 = matriz_3x5.reshape((5, 3)) # Ou meu_vetor_15.reshape((5,3))
print(f"\nMatriz 5x3 (forma {matriz_5x3.shape}):\n{matriz_5x3}")


# Célula de Markdown
"""
---

### 3. Achatar Arrays (`.flatten()` ou `.ravel()`)

Para converter um array multidimensional de volta para um array unidimensional, podemos usar `.flatten()` ou `.ravel()`.

* `.flatten()`: Retorna uma **cópia** do array achatado.
* `.ravel()`: Retorna uma **vista** do array achatado (se possível). Se o array original for modificado, a vista também será. Geralmente mais rápido.

**Exemplo:**
"""

# Célula de Código
matriz_original_flat = np.array([
    [1, 2, 3],
    [4, 5, 6]
])
print(f"Matriz original:\n{matriz_original_flat}")

# Usando flatten
array_achatado_flatten = matriz_original_flat.flatten()
print(f"\nArray achatado com flatten (cópia): {array_achatado_flatten}")
array_achatado_flatten[0] = 99 # Modificar a cópia
print(f"Array achatado modificado: {array_achatado_flatten}")
print(f"Matriz original (não modificada): \n{matriz_original_flat}")

# Usando ravel
array_achatado_ravel = matriz_original_flat.ravel()
print(f"\nArray achatado com ravel (vista): {array_achatado_ravel}")
array_achatado_ravel[0] = 88 # Modificar a vista
print(f"Array achatado modificado: {array_achatado_ravel}")
print(f"Matriz original (modificada!): \n{matriz_original_flat}")

# Célula de Markdown
"""
**Experimentem:**

1.  Criem uma matriz 2x4.
2.  Achate-a usando `.flatten()` e imprimam o resultado.
3.  Achate-a usando `.ravel()` e imprimam o resultado.
"""

# Célula de Código (para o exercício)
minha_matriz_2x4 = np.array([
    [1, 2, 3, 4],
    [5, 6, 7, 8]
])
print(f"Minha matriz 2x4:\n{minha_matriz_2x4}")

achatado_flat = minha_matriz_2x4.flatten()
print(f"\nAchatado com flatten: {achatado_flat}")

achatado_ravel = minha_matriz_2x4.ravel()
print(f"Achatado com ravel: {achatado_ravel}")


# Célula de Markdown
"""
---

### 4. Adicionar/Remover Dimensões (`np.newaxis`, `.squeeze()`)

Às vezes, precisamos de adicionar uma nova dimensão a um array (por exemplo, para que ele seja compatível com outro array numa operação).

* `np.newaxis`: Adiciona uma nova dimensão de tamanho 1.
* `.squeeze()`: Remove dimensões de tamanho 1.

**Exemplo:**
"""

# Célula de Código
vetor_original = np.array([1, 2, 3])
print(f"Vetor original (forma {vetor_original.shape}): {vetor_original}")

# Adicionar uma dimensão para que seja uma matriz coluna (3 linhas, 1 coluna)
vetor_coluna = vetor_original[:, np.newaxis]
print(f"\nVetor como coluna (forma {vetor_coluna.shape}):\n{vetor_coluna}")

# Adicionar uma dimensão para que seja uma matriz linha (1 linha, 3 colunas)
vetor_linha = vetor_original[np.newaxis, :]
print(f"\nVetor como linha (forma {vetor_linha.shape}):\n{vetor_linha}")

# Remover dimensões de tamanho 1
matriz_com_dim_extra = np.array([[1], [2], [3]])
print(f"\nMatriz com dimensão extra (forma {matriz_com_dim_extra.shape}):\n{matriz_com_dim_extra}")
matriz_espremida = matriz_com_dim_extra.squeeze()
print(f"Matriz espremida (forma {matriz_espremida.shape}): {matriz_espremida}")

**Experimentem:**

2.  Transformem-no num array 2D com uma única coluna (forma `(3, 1)`).
3.  Transformem-no num array 2D com uma única linha (forma `(1, 3)`).
"""

# Célula de Código (para o exercício)

print(f"Dados 1D (forma {dados_1d.shape}): {dados_1d}")

print(f"\nDados como coluna (forma {dados_coluna.shape}):\n{dados_coluna}")

dados_linha = dados_1d[np.newaxis, :]
print(f"\nDados como linha (forma {dados_linha.shape}):\n{dados_linha}")
