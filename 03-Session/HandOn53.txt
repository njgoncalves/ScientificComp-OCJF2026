# Célula de Markdown
"""
---

### Exercício 1 (Matemática): Operações com Vetores e Matrizes

**Cenário:** Temos dois vetores e uma matriz.

* Vetor A: `[1, 2, 3]`
* Vetor B: `[4, 5, 6]`
* Matriz M: `[[1, 0], [0, 1]]` (Matriz Identidade 2x2)

**Tarefa:**
1.  Criem os arrays NumPy para o Vetor A, Vetor B e Matriz M.
2.  Calculem a soma elemento a elemento de Vetor A e Vetor B.
3.  Calculem o produto elemento a elemento de Vetor A e Vetor B.
4.  Calculem o produto escalar (dot product) de Vetor A e Vetor B (usem `np.dot()` ou `@`).
5.  Multipliquem a Matriz M por um escalar `5`.
6.  Transponham a Matriz M (embora seja simétrica, pratiquem o método).
"""

# Célula de Código (para o Exercício 1)
vetor_a = np.array([1, 2, 3])
vetor_b = np.array([4, 5, 6])
matriz_m = np.array([[1, 0], [0, 1]])

print(f"Vetor A: {vetor_a}")
print(f"Vetor B: {vetor_b}")
print(f"Matriz M:\n{matriz_m}")

# 2. Soma elemento a elemento
soma_vetores = vetor_a + vetor_b
print(f"\nSoma de A + B: {soma_vetores}")

# 3. Produto elemento a elemento
produto_elementos = vetor_a * vetor_b
print(f"Produto elemento a elemento de A * B: {produto_elementos}")

# 4. Produto escalar (dot product)
produto_escalar = np.dot(vetor_a, vetor_b) # ou vetor_a @ vetor_b
print(f"Produto escalar de A . B: {produto_escalar}")

# 5. Multiplicar Matriz M por escalar
matriz_m_escalar = matriz_m * 5
print(f"\nMatriz M * 5:\n{matriz_m_escalar}")

# 6. Transpor Matriz M
matriz_m_transposta = matriz_m.T
print(f"\nMatriz M Transposta:\n{matriz_m_transposta}")


# Célula de Markdown
"""
---

### Exercício 2 (Físico-Química): Análise de Dados de Reação Química

**Cenário:** Temos uma série de medições da concentração de um reagente ao longo do tempo, em milimoles/litro (mmol/L).

`concentracao_mmol_l = np.array([100, 85, 72, 60, 50, 42, 35, 30])`

**Tarefa:**
1.  Converta a concentração de mmol/L para moles/L (divida por 1000). Guarde num novo array.
2.  Calcule a concentração média em moles/L.
3.  Encontre a concentração mínima e máxima em moles/L.
4.  Determine quantos pontos de medição têm concentração abaixo de `0.06 moles/L`.
"""

# Célula de Código (para o Exercício 2)
concentracao_mmol_l = np.array([100, 85, 72, 60, 50, 42, 35, 30])
print(f"Concentração em mmol/L: {concentracao_mmol_l}")

# 1. Converter para moles/L
concentracao_mol_l = concentracao_mmol_l / 1000
print(f"Concentração em mol/L: {concentracao_mol_l}")

# 2. Concentração média
media_concentracao = np.mean(concentracao_mol_l)
print(f"Concentração média em mol/L: {media_concentracao:.4f}")

# 3. Concentração mínima e máxima
min_concentracao = np.min(concentracao_mol_l)
max_concentracao = np.max(concentracao_mol_l)
print(f"Concentração mínima em mol/L: {min_concentracao}")
print(f"Concentração máxima em mol/L: {max_concentracao}")

# 4. Pontos abaixo de 0.06 mol/L
pontos_abaixo_0_06 = concentracao_mol_l[concentracao_mol_l < 0.06]
num_pontos_abaixo = len(pontos_abaixo_0_06) # ou np.sum(concentracao_mol_l < 0.06)
print(f"Número de pontos com concentração abaixo de 0.06 mol/L: {num_pontos_abaixo}")

"""
---

### Exercício 3 (Biologia e Geologia): Análise de Dados de Terreno

**Cenário:** Temos uma matriz que representa a elevação (em metros) de um terreno numa grelha 3x3.

`elevacoes = np.array([[150, 160, 155], [170, 180, 165], [160, 175, 170]])`

**Tarefa:**
1.  Criem o array NumPy para as elevações.
2.  Encontrem a elevação máxima e mínima em todo o terreno.
3.  Calculem a elevação média de cada linha (média de elevação em cada "fila" de terreno).
4.  Calculem a elevação média de cada coluna (média de elevação em cada "coluna" de terreno).
5.  Identifiquem todas as áreas (elementos) com elevação superior a `170 metros`.
"""

# Célula de Código (para o Exercício 3)
elevacoes = np.array([
    [150, 160, 155],
    [170, 180, 165],
    [160, 175, 170]
])
print(f"Elevações do terreno:\n{elevacoes}")

# 2. Elevação máxima e mínima
elevacao_max = np.max(elevacoes)
elevacao_min = np.min(elevacoes)
print(f"\nElevação máxima: {elevacao_max} metros")
print(f"Elevação mínima: {elevacao_min} metros")

# 3. Elevação média de cada linha (axis=1)
media_por_linha = np.mean(elevacoes, axis=1)
print(f"Elevação média por linha: {media_por_linha:.2f} metros")

# 4. Elevação média de cada coluna (axis=0)
media_por_coluna = np.mean(elevacoes, axis=0)
print(f"Elevação média por coluna: {media_por_coluna:.2f} metros")

# 5. Áreas com elevação superior a 170 metros
areas_altas = elevacoes[elevacoes > 170]
print(f"Áreas com elevação superior a 170 metros: {areas_altas} metros")

"""
---

### Exercício 4 (Matemática/Física): Simulação de Posição de Projétil

**Cenário:** Um projétil é lançado e a sua posição horizontal (`x`) e vertical (`y`) ao longo do tempo são dadas por dois arrays.

* `tempo = np.linspace(0, 10, 11)` (0 a 10 segundos, 11 pontos)
* `velocidade_inicial_x = 10` (m/s)
* `velocidade_inicial_y = 20` (m/s)
* `gravidade = 9.8` (m/s^2)

Fórmulas de movimento:
* $x(t) = velocidade\_inicial\_x * t$
* $y(t) = velocidade\_inicial\_y * t - 0.5 * gravidade * t^2$

**Tarefa:**
1.  Criem o array `tempo` usando `np.linspace`.
2.  Calculem o array de posições `x` para cada ponto no tempo.
3.  Calculem o array de posições `y` para cada ponto no tempo.
4.  Combinem os arrays `x` e `y` numa única matriz 2D onde a primeira coluna é `x` e a segunda coluna é `y`. A forma final deve ser `(11, 2)`.
5.  Imprimam a matriz de posições.
"""

# Célula de Código (para o Exercício 4)
velocidade_inicial_x = 10
velocidade_inicial_y = 20
gravidade = 9.8

# 1. Criar array tempo
tempo = np.linspace(0, 10, 11)
print(f"Array de tempo:\n{tempo}")

# 2. Calcular posições x
posicao_x = velocidade_inicial_x * tempo
print(f"\nArray de posições X:\n{posicao_x}")

# 3. Calcular posições y
posicao_y = (velocidade_inicial_y * tempo) - (0.5 * gravidade * (tempo ** 2))
print(f"Array de posições Y:\n{posicao_y}")

# 4. Combinar x e y numa matriz (11, 2)
# Podemos usar np.column_stack ou np.stack com axis=1
posicoes_projeteis = np.column_stack((posicao_x, posicao_y))
# Ou: posicoes_projeteis = np.stack((posicao_x, posicao_y), axis=1)

# 5. Imprimir a matriz de posições
print(f"\nMatriz de Posições (X, Y) do Projétil (forma {posicoes_projeteis.shape}):\n{posicoes_projeteis}")


# Célula de Markdown
"""
---

### Exercício 5 (Físico-Química/Biologia): Normalização de Dados de Experiência

**Cenário:** Temos um conjunto de medições de um sensor, e queremos normalizá-los para um intervalo entre 0 e 1. A normalização min-max é dada por:
$X_{normalizado} = (X - X_{min}) / (X_{max} - X_{min})$

`dados_sensor = np.array([25, 30, 22, 35, 28, 40, 20])`

**Tarefa:**
1.  Criem o array NumPy `dados_sensor`.
2.  Encontrem o valor mínimo (`X_min`) e máximo (`X_max`) do array.
3.  Calculem o array `dados_normalizados` aplicando a fórmula de normalização.
4.  Imprimam os dados originais e os dados normalizados.
"""

# Célula de Código (para o Exercício 5)
dados_sensor = np.array([25, 30, 22, 35, 28, 40, 20])
print(f"Dados originais do sensor: {dados_sensor}")

# 2. Encontrar min e max
x_min = np.min(dados_sensor)
x_max = np.max(dados_sensor)
print(f"Valor mínimo (X_min): {x_min}")
print(f"Valor máximo (X_max): {x_max}")

# 3. Calcular dados normalizados
# Certificar-se de que X_max - X_min não é zero para evitar divisão por zero
if (x_max - x_min) == 0:
    dados_normalizados = np.zeros_like(dados_sensor) # Se todos os valores forem iguais, normaliza para zero
    print("Todos os valores são iguais, normalizados para zero.")
else:
    dados_normalizados = (dados_sensor - x_min) / (x_max - x_min)

# 4. Imprimir
print(f"Dados normalizados (0 a 1): {dados_normalizados}")
